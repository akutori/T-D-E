@startuml

    state 検索結果:do/検索結果の表示

    検索結果 --> 検索結果 :検索情報が入力\nor\n検索ボタンが押された場合
    検索結果 -left-> mainpage :サイトロゴをクリック

    state タブヘッダー{
        state ifinput <<choice>>
        state "タブ" as tab <<sdlreceive>>

        ifinput --> 検索結果 :ジャンル\n・\nハードをクリック
        ifinput -> cart :カートをクリック
        ifinput -up-> profile :プロフィールアイコンorユーザー名をクリック
    }

}
